### Description
- This project is a role add-on for [**Nebula on the Ship**](https://github.com/Dolly1016/Nebula), adding new roles and commands.

> [!NOTE]
> Some roles in this project are AI-generated. See the [source document](ATTRIBUTIONS.md) for details.

### Installation
1. Download and correctly install the [Nebula on the Ship](https://github.com/Dolly1016/Nebula) mod, and run it successfully at least once.
2. In the game directory, place this project's compressed file into the newly created **Addons** folder.  
   *Note: Do not extract the compressed file.*
3. If no errors occur, the installation is complete.

> [!note]
> If you need a pre-packaged bundle, please visit [Half-Sugar's Nest](https://github.com/hvtXsvc-skysilk/Half-sugar-s-gift/README.md/#半糖的小窝)
