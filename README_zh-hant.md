### 匯入方法
1. 下載 [Nebula on the Ship](https://github.com/Dolly1016/Nebula) 模組並正確匯入，成功執行一次。
2. 在遊戲目錄下應新建的 **Addons** 目錄中加入本專案的壓縮包檔案。  
   *註：請勿解壓此壓縮包檔案*
3. 若無錯誤，則匯入完成。

> [!note]
> 若需下載懶人包，請前往 [半糖的小窩](https://github.com/hvtXsvc-skysilk/Half-sugar-s-gift/README.md/#半糖的小窩)
